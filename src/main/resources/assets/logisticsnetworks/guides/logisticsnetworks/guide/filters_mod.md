# Mod Filter

Mod Filter checks namespace, like `minecraft` or `mekanism`.

You can type a namespace or select from the extractor picker.

Target type can be:

1. Items
2. Fluids
3. Chemicals

In the current UI flow, you work with one selected mod value.
